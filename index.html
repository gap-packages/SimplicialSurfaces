GAP Package SimplicialSurfaces

The SimplicialSurfaces package provides functionality for working with simplicial surfaces and generalisations

The current version of this package is version 0.8, released on 2026-05-15. For more information, please refer to the package manual. There is also a README file.

License: GPL-3.0-or-later

Dependencies

This package requires GAP version >= 4.12

The following other GAP packages are needed:

  • AttributeScheduler >=0.1
  • Digraphs >=1.1.1

The following additional GAP packages are not required, but suggested:

  • GAPDoc >= 1.6
  • AutoDoc >=2019.05.20
  • IO >=2.2
  • Grape >=4.8.2
  • NautyTracesInterface >=0.2
