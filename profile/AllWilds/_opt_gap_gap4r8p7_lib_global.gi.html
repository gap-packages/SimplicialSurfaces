12##  the global variable namespace, allowing globals to be accessed and
13##  set by name.
14##
15##  This is defined in two stages. global.g defines "capitalized" versions
16##  of the functions which do not use Info or other niceties and are not
17##  set up with InstallGlobalFunction. This can thus be read early, and
18##  the functions it defines can be used to define functions used to read
19##  more of the library.
20##
21##  This file and global.gd   install the really "public"
22##  functions and can be read later (once Info, DeclareGlobalFunction,
23##  etc are there)

26##  variable name contains characters not recognised as part of
