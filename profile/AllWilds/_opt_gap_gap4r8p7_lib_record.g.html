11##  This file contains methods for records.
12##  Compared to &GAP; 3, where records were used to represent domains and
13##  all kinds of external arithmetic objects, in &GAP; 4 there is no
14##  important role for records.
15##  So the standard library provides only methods for `PrintObj', `String',
16##  `=', and `<', and the latter two are not installed to compare records
17##  with objects in other families.
18##
19##  In order to achieve a special behaviour of records as in &GAP; 3 such
20##  that a record can be regarded as equal to objects in other families
21##  or such that a record can be compared via `<' with objects in other
22##  families, one can load the file `compat3c.g'.
